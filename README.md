# Chaitanya Bhujbal — 3D Portfolio

This is my personal 3D portfolio website, built to showcase my work and skills as a Full Stack Developer. It features an interactive 3D scene, smooth animations, and a modern design using React, Three.js, and GSAP.

Live site: [https://chaitanyabhujbal.netlify.app/](https://chaitanyabhujbal.netlify.app/)

![Portfolio Preview](public/images/preview1.png?v=1)

## Features

- **Interactive 3D Scene**: Custom character and environment rendered with React Three Fiber.
- **Smooth Animations**: High-performance transitions and scroll-driven effects powered by GSAP.
- **Modern Tech Stack**: Built with React 18, TypeScript, and Vite for speed and reliability.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **3D Engine**: Three.js, React Three Fiber, React Three Drei
- **Animations**: GSAP (GreenSock Animation Platform)
- **Physics**: Rapier (via React Three Rapier)
- **Styling**: Vanilla CSS

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/chaitanyabhujbal912006-afk/3d-portfolio_.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
